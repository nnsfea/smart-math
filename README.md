## 🎯 Overview

SmartMath is an **interactive educational platform** designed to teach graph mathematics concepts through engaging, gamified learning. Students practice various graph types with multiple difficulty levels, earn points, unlock achievements, and track their progress.

**Language:** Primarily in **Malay (Bahasa Malaysia)** with English translations

---

## ✨ Key Features

### 🎮 Gamification System
- **Point System**: Earn 10 points for each correct answer
- **Real-time Score Tracking**: Live score updates and session management
- **Level Unlocking**: 
  - Level 1: Start at 0 points
  - Level 2: Unlock at 20 points ⭐
  - Level 3: Unlock at 60 points ⭐⭐
- **Performance Feedback**: Personalized messages based on quiz completion

### 📚 Learning Topics
- **Linear Graphs** (Graf Linear) - `y = mx + c` equations and applications
- **Quadratic Graphs** (Graf Kuadratik) - Parabolas, vertex, and roots
- **Exponential Graphs** (Graf Eksponen) - Growth/decay functions
- **Trigonometric Graphs** (Graf Trigonometrik) - Sine, cosine, tangent

### 🎚️ Difficulty Levels
Each topic has 3 difficulty levels:
- **🟢 Mudah (Easy)** - Fundamental concepts
- **🟡 Sederhana (Medium)** - Intermediate problem-solving
- **🔴 Sukar (Hard)** - Advanced applications

### 🎨 Modern UI/UX
- **Smooth Animations**: Fade-in, slide-up, bounce, pulse effects
- **Responsive Design**: Perfect on desktop, tablet, and mobile
- **Visual Progress Indicators**: Progress bars and question counters
- **Interactive Feedback**: Immediate answer validation with effects
- **Gradient Design**: Purple-themed modern interface

## 📚 Topics & Questions

### 1. Linear Graphs (4 Topics)
**Questions per difficulty:** 5 each = 15 total

Easy:
- Finding y values given x
- Simple linear substitution
- Basic equation evaluation

Medium:
- Finding x values given y
- Calculating slope and intercepts
- Equation properties

Hard:
- Point-slope relationships
- Parallel lines
- Intercept calculations

### 2. Quadratic Graphs (4 Topics)
**Questions per difficulty:** 5 each = 15 total

Easy:
- Finding minimum/maximum values
- Basic quadratic evaluation
- Vertex form understanding

Medium:
- Root finding
- Vertex calculations
- Parabolic properties

Hard:
- Complex vertex problems
- Advanced root finding
- Coordinate determination

### 3. Exponential Graphs (4 Topics)
**Questions per difficulty:** 5 each = 15 total

Easy:

- Powers and exponents
- Fundamental calculations

Medium:

- Exponential multipliers

Hard:

Easy:

- Standard positions

Medium:
- Amplitude calculations
- Transformed functions

Hard:

- Advanced trigonometry
